Question 1

public class Main {
    public static void main(String[] args) {
        byte a = "durga";
        System.out.print(a);
    }
}

Topic: Type Conversion

What will be the outcome when you try to compile this program?

  • A. Compilation Error: incompatible types: String cannot be converted to byte
  • B. Compilation Error: incompatible types: possible lossy conversion
  • C. 0
  • D. Exception

Answer: A. Compilation Error: incompatible types: String cannot be converted to byte

Explanation: The code will produce a compilation error because a String cannot be assigned to a byte variable.

Question 4

public class Main {
    public static void main(String[] args) {
        byte a = 128;
        System.out.print(a);
    }
}

Topic: Data Types

What will be the outcome when you try to compile this program?

  • A. Compilation Error: constant expression required
  • B. Compilation Error: possible lossy conversion from int to byte
  • C. 128
  • D. Exception

Answer: B. Compilation Error: possible lossy conversion from int to byte

Explanation: The code will produce a compilation error because 128 is outside the valid range for a byte (-128 to 127). Assigning 128 to a byte variable requires explicit casting to handle the value properly.
